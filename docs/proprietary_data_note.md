# Proprietary Data Note

The FAWT structural mass basis used in the manuscript is partner-provided category-level concept information. The absolute numerical mass inputs are not disclosed in this repository.

The repository includes the non-proprietary derived outputs required to interpret the manuscript:

- central GWP intensity results,
- deterministic-envelope outputs,
- sensitivity-test results,
- figure source data,
- disclosure status labels.

FAWT is treated as a private scenario case, not as a public reference-design benchmark. It is marked separately in the manuscript and interpreted apart from the public/proxy support-structure comparisons.
