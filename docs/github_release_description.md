# GitHub Release Description

Release name: `v1.0.1-submission`

Submission-version archive for the manuscript "Climate-Focused LCA of Wind Turbine Support Structures: Source-Explicit Inventories and Rank Fragility."

This release contains the non-proprietary data tables, scripts, Monte Carlo summaries, deterministic stress-test outputs, and figure source data used to generate the manuscript results. Partner-provided FAWT structural mass inputs are not disclosed; derived scenario outputs and disclosure status are included.
